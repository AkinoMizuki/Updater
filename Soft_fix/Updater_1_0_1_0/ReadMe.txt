;		仕様
;============================================================

Zipのファイル名：
	・メインexe_メジャー_マイナー_ビルド_リビジョン_.zip
例：
	アップデート対象exe名：Updater
	アセンブリバージョン：ver1.0.0.0
	・Updater_1_0_0_0.zip
	
latest.txtの中身はverのみ
	例：
		1.0.0.0

;============================================================
;		コンフィグファイル設定
;============================================================
<?xml version="1.0" encoding="utf-8" ?>
<configuration>
  <WebServer>false</WebServer>
    ⇒ webサーバー = true, ローカルサーバー = false
    
  <IpAddress>43.26.251.71</IpAddress>
    ⇒ IPアドレス or ドメイン
    
  <FilePass>test_library\000_XTT\000_SoftUpdater</FilePass>
    ⇒ IPアドレスを抜いたZipの有るファイル
    
  <ExeNeme>CheckUpload</ExeNeme>
    ⇒ 
    
  <NewRevFile>latest.txt</NewRevFile>
    ⇒ 最終バージョンを基準したテキストファイル名
    
  <KeepFolder></KeepFolder>
    ⇒ アップデート時に残すフォルダー１個まで
    
  <SoftReStart>false</SoftReStart>
    ⇒ アップデート後にアップデート対象ソフトを起動するか。
    
  <PingWait>500</PingWait>
   ⇒ サーバーとの接続確認でpingを送るウェイト(ms)
   
</configuration>
